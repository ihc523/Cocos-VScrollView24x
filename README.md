#### VScrollView

基于 CocosCreater 引擎编辑器的全新滚动组件和虚拟列表,这是 v2.4.x 版本. 3.8.x 版本 在另一个仓库 [3.8.x 仓库地址](https://github.com/soidaken/VScrollView).

功能基本一致,除了

- 分层合批优化, 尝试了将子项的 Label 使用独立相机渲染,渲染效果没有问题，但是 drawcall 没有下降,我也不知道为什么.
- 2.4.x 属于老项目了,暂时支持功能就足够了,你如果确实有 dc 优化的需求,可以参考论坛内一些优化的方式
- 此仓库到此为止,后续也不再同步新的功能.
- 所有目前的案例使用方式和 API 都和 3.8.x 版本完全一致,案例代码可直接覆盖使用.
- [在线预览](https://soidaken.github.io/VSCrollView_SamplesPreView/)
